## 10226 - Hardwood Species
### Isha Perry
### Description:

Each test case of your program consists of a list of the species of every tree observed by the satellite;
one tree per line. No species name exceeds 30 characters. There are no more than 10,000 species and
no more than 1,000,000 trees. For each test case print the name of each species represented in the population, 
in alphabetical order, followed by the percentage of the population it represents, to 4 decimal places.
